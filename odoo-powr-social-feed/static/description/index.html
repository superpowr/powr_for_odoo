POWr Social Feed

Getting Started with POWr:

POWr is easy to install and free to get started. After installing on Odoo, edit any website page and find the POWr Plugins in the left sidebar. Simply drag and drop a Social Feed to add it to the page and click Edit Social Feed to edit.
